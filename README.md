# MomWorld ♡ — Mother’s Day Y2K Website

A nostalgic Y2K-inspired Mother’s Day website created using HTML, CSS, and JavaScript.

This project is designed like an early 2000s personal website with:
- dreamy purple/pink aesthetics
- glossy UI
- floating hearts & sparkles
- animated loading screen
- scrapbook-style photo layout
- custom cursor effects
- background music support

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Folder Structure

mothers-day-site/assets/

## Setup

1. Clone the repository
2. Add your images and music inside the `assets` folder
3. Open `index.html` in your browser

## GitHub Pages Deployment

1. Push the project to GitHub
2. Go to:
   Settings → Pages
3. Under “Source” select:
   `Deploy from branch`
4. Select:
   `main` branch → `/root`
5. Save changes

Your website will be live at:

https://yourusername.github.io/repository-name/
